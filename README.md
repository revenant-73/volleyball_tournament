# Volleyball Tournament Manager

A clean, modern web application for managing volleyball tournaments, pool play, and brackets.

## Features
- **Team Setup**: Easy team entry and seeding.
- **Pool Play**: Automated pool assignments and scoring.
- **Standings**: Real-time standings based on match wins and set ratios.
- **Bracket View**: Dynamic tournament brackets for single-elimination play.
- **Match Results**: Simplified score entry.
